When writing HTML

  1. Use Side-By-Side Windows
  2. Never write more than 5 lines without testing
    • When learning, it's a good idea to never write more than one element without testing
  3. when writing an element, always write the opening tag, then the closing tag, then the content in between.
  4. Use eindentation to structure you code and make it more human-readable.
    • If you are nesting seomthing with an element, follow this process:
      • write the opening tag
      • write the closing tag
      • place a blank lin between the tags (by pressing enter)
      • make sure the opening tag and closing tag are indented the same amount
      • now write the content that goes between.
      • open-close-new line-indent
  5. learn how your editor syntax highlighting works
  6. use an html validator:Validator
